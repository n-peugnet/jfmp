# Jellyfin Music Player

AKA jfmp

## Install

    pip install --user pipenv
    pipenv install

## Run

    pipenv run ./main.py

##  Built With

-   [music-player-core](https://github.com/albertz/music-player-core)
-   [jellyfin-apiclient-python](https://github.com/iwalton3/jellyfin-apiclient-python)
